## 💧waterride(ios)

- 주제 : 주식 추가 매수, 매도 시 평균 단가 계산이 가능한 서비스
- 프로젝트 기간 : 2021.02
- 팀원 : 석정환

## 📚기술 스택
- Swift

## 📝프로젝트 내용
- 기본적인 평균 단가 계산 기능, 실시간으로 반응하도록 로직 구현
- Delegate패턴을 활용해 가독성을 위한 자리 수(콤마) 구분
